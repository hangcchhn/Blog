## RESTful API
> HTTP JSON

- REST(Representational State Transfer):具象状态传输
- `Content-Type:application/json`

---
| 请求方法 | 接口功能 |
| :- | :- |
| POST /user | 创建用户 |
| GET /user | 获取所有用户 |
| GET /user/id | 获取指定用户 |
| PUT /user/id | 更新指定用户全部信息 |
| PATCH /user/id | 更新指定用户部分信息 |
| DELETE /user/id | 删除指定用户 |
