# C语言

## 启动函数

``` c

// 只适用C语言
void main()
{
	// code
} 


// 即适用C语言又适用C++语言
int main(int argc, char *argv[]){
	// code
    return 0;
}
// C语言编译器将可执行文件名称也作为参数
// 命令行中的字符串也包括可执行文件名称
// 整型argc表示命令行中字符串的个数，
// 指针数组argv[]指向命令行中的各个字符串

```
