

--------------------------------------------------------------------------------------------------
用户操作
useradd username

-s /sbin/nologin
-s /bin/bash

-d /home/username

usermod -G groupname username

userdel username

-------------------------------------------
组操作

groupadd groupname

groupmod groupname

groupdel groupname



-------------------------------------------
密码设置

passwd username
输入password
输入password

相当于指定用户名和密码进行修改
echo username:password|chpasswd


-------------------------------------------
给文件或文件夹设置用户和组

-R 或 -r 递归regression

chown -R groupname:username /home/username


--------------------------------------------------------------------------------------------------
给文件或文件夹设置权限（包括用户和组，读写和执行两大类）


linux
chmod -r [ ] path

mac
chmod -R [ ] path
