centos：
查看所所有安装的内核版本
rpm -qa| grep kernel
删除具体单个内核
rpm -e 具体内核名和版本

查看所所有安装的内核版本
yum search kernel
删除具体单个内核
yum remove 具体内核名和版本

ubuntu:
查看已安装内核
dpkg --get-selections |grep linux-image

卸载内核
apt remove linux-image-?

卸载内核并删除配置文件
apt remove --purge linux-image-?

移除失效的启动项
update-grub



apt autoclean
apt clean

yum install yum-utils
package-cleanup --oldkernels --count=1
