## Unix
- 二进制安装包
```
# 离线安装
```

## Linux

### RedHat/CentOS
> rpm(RedHat Package Manager)
- 离线安装:rpm(RedHat Package Manager)

- 在线安装:yum(Yellowdog Updater Modified)



### Debian/Ubuntu
> dpkg(Debian Package) /deb(Debian)

- 离线安装:dpkg(Debian Package)

- 在线安装:apt(Advanced Packaging Tool)
