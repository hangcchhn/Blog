
列出所有容器版本
wsl --help

--list
    --quiet 名称信息
    --verbose 详细信息

--shutdown 关闭容器

    

wsl -l -v



  NAME                   STATE           VERSION
* Ubuntu                 Running         1


指定容器版本
wsl --set-version Ubuntu 2

升级默认容器版本
wsl --set-default-version 2

默认分发
wsl --set-default Ubuntu

wsl --status
wsl --update


wsl --list --online

wsl --install --distribution Ubuntu
