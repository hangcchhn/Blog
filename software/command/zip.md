


## 解压压缩

```


# 解压x：抽取extract
tar -xvf xxx.tar
tar -zxvf xxx.tar.gz
tar -jxvf xxx.tar.bz2
tar -Zxvf xxx.tar.Z
tar -Jxvf xxx.tar.xz

unzip xxx.zip
unrar e xxx.rar

xz -d xxx.tar.xz
# 解压xz文件生成tar文件

# 压缩c：创建create
tar -cvf xxx.tar xxx
tar -zcvf xxx.tar.gz xxx
tar -jcvf xxx.tar.bz2 xxx
tar -Zcvf xxx.tar.Z xxx
tar -Jcvf xxx.tar.xz xxx



```
