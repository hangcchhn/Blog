# Docker网络
> network


- Docker在主机上虚拟一个虚拟网桥，在网卡网段内分配一个IP给容器
- 外部主机访问容器：端口映射（将容器端口映射到主机端口）

```
-p
```
---

## 网络模式

- 桥接模式`-net=bridge`
    - 默认的网络模式
    - 容器可以访问，主机可以访问，外部主机不能访问
    - 容器可以通过主机的NAT规则访问外部网络
- 主机模式`-net=host`
    - vmware虚拟机网络中的NAT模式

- 容器模式`-net=container:`



- `-net=none`
    - 只有回环网卡lo，只能访问localhost(127.0.0.1)
