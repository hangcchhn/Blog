Throwable：Java语言中所有错误或异常的超类。
下一层分为Error和Exception


错误Error:系统内部错误和资源耗尽错误
异常Exception又有两个分支：
1.运行时异常RuntimeException： IOException、SQLException

2.检查异常CheckedException：


Error是系统级别的，程序无法处理
Exception是程序级别的，需要程序处理


捕获异常并针对性处理：
try {
    // code
}catch(Exception exception){

}


Stream stream = null
try {
    stream = new Stream();
    // code

}catch(Exception exception){

}finally{
    if (stream != null){
        try {
            stream.close();
        }catch(Exception exception){

        }
    }
}

流对应的类都实现了自动关闭接口AutoCloseable
stream:创建流对象语句，如果多个，使用';'隔开
此语句总会执行语句执行对非空流对象进行关闭操作
try (stream) {
    // code

}catch(Exception exception){

}



抛出异常的两种方式：

1.用在函数上:可以跟多个
public return method(Class param) throws Exception {
    //code
}

2.用在函数内:只能跟一个
throw new Exception();
